## 游戏概述

《深渊征途》是一款类 Slay the Spire 的单机肉鸽卡牌游戏。你将选择一名冒险者，沿随机生成的地图向上攀登，经历战斗、事件、商店，最终挑战每幕的 Boss。**死亡是永久的**——每次失败都要重新开始，但随机性确保每局体验不同。

**目标**：通关三幕，击败最终 Boss。

---

## 基本流程

```
选择职业
    ↓
第一幕地图（7层 + Boss）
    ↓
Boss 遗物选择
    ↓
第二幕地图（7层 + Boss）
    ↓
Boss 遗物选择
    ↓
第三幕地图（5层 + Boss）
    ↓
击败最终 Boss → 胜利！
```

每层选择一个节点进入，不同节点提供不同内容。选择相邻节点后才可进入下一层。

## 职业选择

### ⚔️ 战士（Warrior）

| 属性     | 数值         |
|----------|--------------|
| 初始 HP  | 80           |
| 初始能量 | 3            |
| 起始遗物 | 燃烧之血     |
| 起始牌   | 猛击×5、格挡×4、重击×1 |

**风格**：强力格挡 + 力量叠加，适合硬扛。燃烧之血每场战斗结束回复 6 HP（Boss遗物升级后回复 12 HP）。

**核心牌组方向**：
- 力量叠加流：激怒 + 极限爆发 + 恶魔形态，叠到力量后普通攻击也能打出高伤害
- 格挡反击流：壁垒 + 重拳（等于格挡值伤害）+ 火焰屏障（受击反弹）
- AoE 流：横扫 + 燃烧牺牲 + 死亡镰刀（AoE 回血）

## 战斗系统

### 回合流程

```
玩家回合开始
  → 遗物触发（白兽雕像回血、战争艺术能量、法球被动）
  → 抽5张牌
  → 使用能量打出卡牌
  → 按 E 或点击【结束回合】
  → 弃手牌（除非有符文金字塔遗物）
  → 格挡清零（除非有壁垒牌/卡尺遗物）
敌人回合
  → 敌人依次行动（攻击/格挡/施加状态）
  → 毒素/灼伤持续伤害结算
```

### 伤害计算

```
实际伤害 = 基础伤害 × (1 + 力量 × 0.1) × 虚弱系数 × 易伤系数
虚弱系数 = 0.75（玩家被虚弱时攻击打折）
易伤系数 = 1.5（敌人被易伤时受到额外伤害）

受到伤害时先扣格挡，格挡不足时扣HP
```

### 状态效果

| 状态     | 图标 | 效果                           |
|----------|------|--------------------------------|
| 力量     | 💪   | 每点力量提升攻击伤害            |
| 敏捷     | 🤸   | 每点敏捷提升格挡值              |
| 虚弱     | 💔   | 攻击伤害降低25%，持续N回合      |
| 易伤     | ⬇️   | 受到伤害增加50%，持续N回合      |
| 毒素     | ☠️   | 每回合损失N点HP，N每回合-1      |
| 灼伤     | 🔥   | 回合结束损失1点HP（诅咒牌）     |
| 回声形态 | 🔮   | 每回合首张非能力牌打出两次      |

### 战斗奖励

- 击败普通敌人：金币 + 选择1张牌（3选1）+ 概率获得药水
- 击败精英：遗物 + 金币 + 牌
- 击败Boss：选择Boss遗物（3选1）

## 法球系统（法师）

法师专属机制，最多可持有3个法球槽（可通过能力牌扩展）。

### 引导法球

打出相关卡牌时自动引导法球。若法球槽已满，最旧的法球会被**激活**后移除（溢出触发）。

### 法球类型

| 法球     | 图标 | 被动效果（每回合） | 激活效果         |
|----------|------|--------------------|------------------|
| 闪电法球 | ⚡   | 对随机敌人3点伤害  | 对随机敌人8点伤害 |
| 冰霜法球 | ❄️   | 获得2点格挡        | 获得5点格挡      |
| 等离子体 | 🔵   | 获得1点能量        | 获得2点能量      |

### 激活方式

1. **双重施法**：激活当前（最旧）法球2次
2. **法球槽溢出**：引导第4个法球时，最旧的自动激活
3. **卡牌特效**：部分卡牌会直接激活

### 相关遗物

- **破裂核心**（起始）：每场战斗开始自动引导1个闪电法球
- **冰封核心**：回合结束时若法球槽为空，自动引导冰霜法球

## 升级与强化

### 篝火升级

在🔥节点可选择升级1张手牌。升级效果：
- 攻击牌：伤害 +30% + 2
- 技能牌：格挡/效果 +30% + 2
- 费用 > 0 的牌费用 -1
- 升级后牌名后显示 `+`

### 商店移除

在🛒节点可花费金币从牌组中永久移除1张牌（通常100金币），有利于精简牌组。

## Boss 攻略

### 第一幕 Boss：卫兵长（Act 1 Boss）

- 规律性地交替攻击和施加虚弱
- 建议：携带移除虚弱的手段，或先积累大量格挡

### 第二幕 Boss：拉格维林（Lagavulin）

⚠️ **特殊机制**：
- **沉眠**：前几回合不行动，期间给玩家叠状态
- **苏醒**：苏醒时立即削弱玩家力量和敏捷各2点
- **吸血**：攻击时同时偷取玩家1点力量和1点敏捷

**对策**：尽量在苏醒前打出足够伤害，或带足够多的回复手段。

### 第三幕 Boss：腐化之心（Corrupt Heart）

⚠️ **高难特殊机制**：
- **无敌**：前4回合无法被伤害
- **治愈**：使用BUFF行动时恢复100 HP
- **诅咒风暴**：使用SPECIAL行动时向玩家弃牌堆加入10张创伤牌
- **每回合攻击**：常规攻击伤害极高

**对策**：
1. 前4回合专注积累格挡和状态，不要浪费高伤手牌
2. 关注其行动意图，在可能触发SPECIAL前准备好手牌
3. 建议携带能够消耗/移除诅咒的手段
4. 毒素流对此Boss有奇效（无敌期间毒素仍然生效）

## 新手建议

### 牌组构建原则

1. **精而不杂**：每次选牌不必非拿不可，牌组越小循环越快
2. **平衡攻防**：纯攻击撑不过后期，纯防御无法击败Boss
3. **注意费用**：高费牌强力但占能量，低费牌灵活但效果弱
4. **专注方向**：尽早确定核心战略（力量叠加、毒素、法球等），避免杂而不精

### 资源管理

- **金币**：优先移除弱牌，再考虑购买遗物，最后买牌
- **药水**：Boss战前保留1-2个强力药水
- **HP**：低于40%时优先寻找篝火，不要为了贪节点送命

### 路线选择

- **第一幕**：前几层走普通战斗练习，第4-5层可考虑精英（配合标本昆虫遗物）
- **第二幕**：商店必去（移除弱牌），篝火最好走一次
- **第三幕**：路线短但节点强度大，保留足够HP迎战最终Boss
